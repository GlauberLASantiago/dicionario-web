🎨 Componentes Visuais Principais

Container (Contêiner)

Caixas que envolvem e organizam outros elementos. É como uma "caixa invisível" que agrupa conteúdo.

Exemplo:

Este é um Container!

Contém múltiplos elementos organizados

<div class="container"> <h2>Título</h2> <p>Conteúdo aqui</p> </div>

Modal / Modal Dialog (Janela Modal)

Aquela "caixinha que aparece no canto" - uma janela flutuante que bloqueia a página de fundo.

Clique no botão para ver um exemplo de Modal em ação!

Toast / Notificação

Mensagem pequena que aparece geralmente no canto/topo e desaparece automaticamente.

Overlay (Sobreposição)

Camada translúcida que fica sobre tudo. Veja no Modal aberto!

O fundo escuro do Modal é um Overlay! 👆

Sidebar (Barra Lateral)

Painel lateral que organiza navegação ou conteúdo. Veja à esquerda da página!

Esse menu à esquerda é a Sidebar! 👈

Header (Cabeçalho)

Seção superior da página. Contém logo, navegação, títulos.

O topo roxo desta página é o Header! ⬆️

💻 Conceitos de Código

Class / Classe

Conjunto de estilos que pode ser reutilizado

.btn-primary { background: #667eea; color: white; padding: 12px 24px; } <button class="btn-primary">Clique</button>

ID / Identificador

Nome único para um elemento específico

#modal { display: none; background: white; } <div id="modal">...</div>

Selector / Seletor

Forma de escolher elementos em CSS

.classe { } /* Seletor de classe */ #id { } /* Seletor de ID */ p { } /* Seletor de elemento */

Property / Propriedade

Atributo de estilo

{ color: blue; /* Propriedade: color */ font-size: 16px; /* Propriedade: font-size */ margin: 10px; /* Propriedade: margin */ }

Value / Valor

O que você coloca em uma propriedade

{ color: blue; /* Valor: blue */ font-size: 16px; /* Valor: 16px */ }

Event / Evento

Ação do usuário (clique, digitação, hover)

<button onclick="handleClick()">Clique</button> <input onchange="handleChange()"> <div onmouseover="handleHover()">...</div>

DOM / Document Object Model

Estrutura hierárquica da página HTML

// Acessar elementos no DOM document.getElementById('id'); document.querySelector('.classe'); document.querySelectorAll('p');
