💻 Conceitos de Código

Class / Classe

Conjunto de estilos que pode ser reutilizado

.btn-primary { background: #667eea; color: white; padding: 12px 24px; } <button class="btn-primary">Clique</button>

ID / Identificador

Nome único para um elemento específico

#modal { display: none; background: white; } <div id="modal">...</div>

Selector / Seletor

Forma de escolher elementos em CSS

.classe { } /* Seletor de classe */ #id { } /* Seletor de ID */ p { } /* Seletor de elemento */

Property / Propriedade

Atributo de estilo

{ color: blue; /* Propriedade: color */ font-size: 16px; /* Propriedade: font-size */ margin: 10px; /* Propriedade: margin */ }

Value / Valor

O que você coloca em uma propriedade

{ color: blue; /* Valor: blue */ font-size: 16px; /* Valor: 16px */ }

Event / Evento

Ação do usuário (clique, digitação, hover)

<button onclick="handleClick()">Clique</button> <input onchange="handleChange()"> <div onmouseover="handleHover()">...</div>

DOM / Document Object Model

Estrutura hierárquica da página HTML

// Acessar elementos no DOM document.getElementById('id'); document.querySelector('.classe'); document.querySelectorAll('p');

🔍 SEO & HTML Semântico

Sitemap.xml (Mapa do Site)

Arquivo que lista todas as páginas do site para o Google saber que elas existem. É o oposto do robots.txt.

<!-- sitemap.xml --> <?xml version="1.0" encoding="UTF-8"?> <urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"> <url> <loc>https://meusite.com/</loc> </url> <url> <loc>https://meusite.com/sobre</loc> </url> <url> <loc>https://meusite.com/contato</loc> </url> </urlset>

Favicon

Aquele ícone pequeno que aparece na aba do navegador, identificando o site visualmente.

<!-- No <head> do HTML --> <link rel="icon" type="image/png" href="/favicon.png"> <!-- Ou para múltiplos tamanhos: --> <link rel="icon" sizes="16x16" href="/favicon-16x16.png"> <link rel="icon" sizes="32x32" href="/favicon-32x32.png"> <link rel="apple-touch-icon" href="/apple-touch-icon.png">
Exemplo:

O ícone 📚 que aparece na aba deste dicionário é um favicon!

Meta Description

O texto que aparece logo abaixo do título do site nos resultados de busca do Google.

<!-- No <head> do HTML --> <meta name="description" content="Aprenda termos web com exemplos interativos. Dicionário visual de HTML, CSS e JavaScript."> <!-- Resultado no Google: --> 📄 Dicionário de Termos Web Aprenda termos web com exemplos interativos. Dicionário visual de HTML, CSS e JavaScript.

Semantic HTML / HTML Semântico

Usar tags que descrevem o significado do conteúdo, não apenas sua aparência. Ajuda o Google e leitores de tela a entenderem a página.

<!-- ❌ Sem semântica (só divs): --> <div class="header">...</div> <div class="nav">...</div> <div class="content">...</div> <div class="footer">...</div> <!-- ✅ Com semântica: --> <header>...</header> <nav>...</nav> <main> <article>...</article> <aside>...</aside> </main> <footer>...</footer>

📍 Posicionamento CSS

Position (Posicionamento)

Define como um elemento é posicionado na página. Os valores principais são: static, relative, absolute, fixed e sticky.

Exemplo visual:
static (padrão)
relative (top:10, left:20)
absolute (top:10, right:10)
fixed vs sticky:

fixed — Fica fixo na tela mesmo ao rolar (como um chat flutuante)

sticky — Gruda no topo ao rolar até ele (como cabeçalhos de tabela)

/* static: posição normal (padrão) */ .element { position: static; } /* relative: move a partir da posição original */ .element { position: relative; top: 10px; left: 20px; } /* absolute: posicionado dentro do pai mais próximo com position */ .element { position: absolute; top: 0; right: 0; } /* fixed: fixo na viewport (não se move ao rolar) */ .element { position: fixed; bottom: 20px; right: 20px; } /* sticky: gruda ao rolar */ .element { position: sticky; top: 0; }
